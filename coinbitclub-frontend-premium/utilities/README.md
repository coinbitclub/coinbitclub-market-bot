# Utilitários e Scripts

Este diretório contém scripts utilitários e ferramentas de desenvolvimento que não fazem parte do código operacional principal.

## Estrutura

### `/scripts/`
Scripts utilitários para:
- Análise de dados
- Configuração de ambiente
- Build e deploy
- Manutenção e limpeza
- Restauração de dados

### `/tests/`
Scripts e configurações de teste:
- Scripts de verificação
- Testes automatizados
- Relatórios de teste
- Configurações de teste (Playwright, Jest, etc.)

## Data de Organização
28 de Julho de 2025

## Objetivo
Separar scripts utilitários do código operacional principal para melhor organização.
