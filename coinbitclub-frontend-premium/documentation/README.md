# Documentação do CoinBitClub Market Bot

## Estrutura

### `/guides/`
Contém guias de configuração, setup e implementação:
- Configuração de variáveis de ambiente
- Setup do projeto
- Guias de integração
- Documentação de features

### `/reports/`
Contém relatórios de implementação e progresso:
- Relatórios de entrega de dashboards
- Relatórios de fases do projeto
- Resumos executivos

### `/deployment/`
Contém scripts e documentação de deploy:
- Scripts de deployment para diferentes ambientes
- Configurações de CI/CD
- Guias de deploy
